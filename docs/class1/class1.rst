Class 1: Introduction to F5 Application Study Tool
==================================================

Welcome! This class will serve as an introduction to the F5 Application Study Tool (AST).

.. note:: This lab guide is meant to provide a hands-on exercise for folks wanting to learn more about and become familiar with the F5 AST. The `public GitHub repo <https://github.com/f5devcentral/application-study-tool>`_ and `Docsite <https://f5devcentral.github.io/application-study-tool/>`_ focus primarily on the administrative/configuration side of the F5 AST, and will be covered at a high level.
   
   We highly recommend you explore not only this lab, but also the repo and docsite!
